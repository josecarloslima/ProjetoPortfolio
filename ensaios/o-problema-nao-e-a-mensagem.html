◇ HIPÓTESE

O problema não é a mensagem.

Muitas vezes a mensagem está correta. O sistema de comunicação é que não está.

Vamos começar colocando, para ser justo, um "quando" em nossa hipótese: Quando o problema não é a mensagem.

Quando uma organização diz que "o público não entendeu", normalmente procura corrigir o texto, alterar uma campanha ou reformular um discurso.

Mas raramente investiga a estrutura que produz esse discurso.

Comunicação acontece em sistemas.
Atendimento.
Interface.
Cultura.
Processos.
Liderança.
Narrativa.

Quando esses elementos não conversam entre si, nenhuma mensagem consegue compensar a incoerência.

Quando o problema não é a frase, quase sempre é o ambiente em que ela foi produzida. Então cabe uma investigação ao ambiente em busca desta "conversa" entre os elementos do grupo. Veja não é sobre frase é sobre contextos.

Esta hipótese permanece aberta.
